4.3绘制图像

drawImage方法可以在画布上绘制图像。drawlmage 方法有三种用法:

    ✧drawlmage(imgx,y): 绘制定位的图像;
    ✧draw lmage(imgx,y,width,height):绘制定位的指定宽度、高度的图像；
    ✧drawlmage(img.sx. swidth,sheig.xyheigh,width):绘制定位剪切的指定宽度、高度的图像。
绘制定位的图像； 创建HTML文件，代码如下：
<!doctype html>
<html>
<head>
<meta charset="utf-8">
<title>应用图像</title>
</head>
<body>
<canvas id="mycanvas" width="300" height="150" style="border:1px solid #d3d3d3">
</canvas>
<script>
var c= document.getElementById("mycanvas");
var cxt= c.getContext("2d");
var img= new Image();
img.onload = function(){
cxt.drawImage(img,50,25,200,100);
}
img.src = "图片/head.jpg";
</script>
</body>
</head>
