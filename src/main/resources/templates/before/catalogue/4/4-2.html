4.2绘制文字

canvas中绘制文字可以使用fillText方法和strokeText方法，这两个方法分别用于以轮廓方式绘制文字。

fillText语法如下:

    filltextxy[maxWidth])

其中，text表示要绘制的文字，x、y表示起始坐标，[maxWidth]表示最大宽度，是可选参数。

以填充方式绘制带渐变效果的文字。
创建HTML文件4.2.1.html，代码如下：

<!doctype html>
<html>
<head>
<meta charset="utf-8">
<title>绘制文字</title>
</head>
<body>
<canvas id="mycanvas" width="300" height="150" style="border:1px solid #d3d3d3">
</canvas>
<script>
var c= document.getElementById("mycanvas");
var cxt= c.getContext("2d");
var gradient = cxt.createLinearGradient(0,0,c.width,0);
gradient.addColorStop(0,"magenta");
gradient.addColorStop(0.5,"blue");
gradient.addColorStop(1,"red");
cxt.fillStyle=gradient;
cxt.font= "30px Verdana";
cxt.fillText("好好学习，天天向上",15,75);
</script>
</body>
</head>
