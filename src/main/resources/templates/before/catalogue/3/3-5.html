3.5新的form属性

HTML5的form表单增加了-些新的属性，以方便开发者控制表单行为。

3.5.1 autocomplete 属性

autocomplete属性用于设置一个表单中的输入域是否拥有自动完成功能，当设置为“on”时，浏览器会根据用户输入的值显示可用的选项，选项的内容是根据用户之前录入的内容确定的。表单的autocomplete属性可以作用于text、Search、Rul、email等表单域。

< form action="#" method="post" autocomplete ="on">

姓名：< input type="text" name="userName"/>

密码：< input type="password" name="userName"/>

< input type="submit" value="提交"/> < /form>

3.5.2 novalidate 属性

novalidate属性是布尔类型的属性，当设置表单属性

为novalidate时，表单提交后，辅入域中输入的值不会被校验，如email类型的表单域不会要求必须包含“@"字母。

< form action="#" method="post" novalidate="novalidate">
电子邮箱：< input type="email" name="txtemail" />
< input type="submit" value="提交"/>
